# Central de Alertas

## Objetivo

Documentar a tela de monitoramento de alertas da rede disponível em **Rede/ Infra**.

## Quando usar

Use esta tela quando for necessário:

- visualizar o gráfico com alerta;
- atualizar os dados exibidos;
- limpar os alertas em aberto.

## Pré-requisitos

- Acesso ao menu **Rede/ Infra > Central de Alertas**.
- Permissão para visualizar e gerenciar alertas.

## Passo a passo

1. Acesse **Rede/ Infra > Central de Alertas**.
2. Revise o conteúdo exibido no gráfico.
3. Clique em **Atualizar** para recarregar os dados.
4. Clique em **Limpar Alertas** para remover os alertas visíveis, se necessário.

## Elementos da tela

| Elemento | Descrição |
|---|---|
| Título do painel | **Gráfico com Alerta**. |
| Botão **Atualizar** | Recarrega as informações do gráfico. |
| Botão **Limpar Alertas** | Limpa os alertas apresentados na tela. |

## Resultado esperado

- O painel de alerta é exibido corretamente.
- O usuário consegue atualizar os dados.
- O usuário consegue limpar os alertas quando a permissão está disponível.

## Observações

- A tela verificada no demo mostra apenas o bloco **Gráfico com Alerta** e os botões **Atualizar** e **Limpar Alertas**.
- Não foram observados outros filtros ou tabelas nesta visualização.

## Dúvidas para revisão

- Há confirmação adicional antes de limpar alertas?
- O gráfico apresenta séries históricas ou apenas o estado corrente?
