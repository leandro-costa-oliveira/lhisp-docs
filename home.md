# LHISP Documentation

Bem-vindo à documentação oficial do **LHISP** — sistema de gestão para provedores de internet.

Esta documentação contém guias passo a passo para operações do módulo **Contratos**, incluindo cadastro de clientes, serviços, acessos e geração de contas.

## Módulo Contratos

| Documento                                                                 | Descrição                                    |
| ------------------------------------------------------------------------- | -------------------------------------------- |
| [Cadastrar Novo Cliente](/contratos/cadastrar-novo-cliente)             | Cadastra um novo cliente/contrato no sistema |
| [Adicionar Serviço Contratado](/contratos/adicionar-servico-contratado) | Vincula um plano de serviço ao contrato      |
| [Adicionar Acesso ao Cliente](/contratos/adicionar-acesso-cliente)      | Configura acesso PPPoE ou IP Fixo            |
| [Gerar Contas do Cliente](/contratos/gerar-contas-cliente)              | Gera e gerencia contas financeiras           |

## Recursos

- **Tema escuro/claro**: Alterne entre os modos usando o toggle no canto superior direito
- **Busca**: Use a barra de busca para encontrar conteúdo rapidamente
- **Navegação instantânea**: As páginas carregam sem recarregar o navegador

## Sobre o LHISP

O LHISP é uma plataforma completa de gestão desenvolvida pela [Altsoft](https://www.altsoft.com.br) para provedores de internet que necessitam de um sistema integrado para gerenciamento de clientes, contratos, serviços e financeiro.
