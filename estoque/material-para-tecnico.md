# Material para Técnico

> **📝 Página-resumo do fluxo fim a fim**
>
> Este guia organiza o processo de material para técnico em páginas menores, com um caminho de leitura sequencial e links para cada etapa.

## Objetivo

Documentar o fluxo completo de entrega de material ao técnico, desde os cadastros-base até a separação e confirmação de entrega.

## Quando usar

Use este mapa quando precisar consultar o fluxo completo ou saltar direto para uma etapa específica.

## Fluxo resumido

1. Cadastrar uma categoria de estoque.
2. Cadastrar um produto.
3. Dar entrada do material.
   - entrada manual em **Almoxarifados**;
   - entrada por **Nota Fiscal de Compra** com XML de teste.
4. Fazer uma **Ordem de Separação** para o técnico.
5. Confirmar a entrega do material na própria ordem.

## Mapa das páginas

| Etapa | Página | Descrição |
|---|---|---|
| 1 | [Categorias de Estoque](/cadastros/estoque/categorias) | Base do cadastro usado no fluxo. |
| 2 | [Produtos](/cadastros/estoque/produtos) | Produto vinculado à categoria de estoque. |
| 3A | [Entrada de Material](/estoque/entrada-de-material) | Caminho manual via **Almoxarifados**. |
| 3B | [Notas Fiscais de Compra](/estoque/notas-fiscais-de-compra) | Caminho fiscal com importação de XML. |
| 4 | [Ordens de Separação](/estoque/ordens-de-separacao) | Criação da ordem para o técnico. |

## Ordem recomendada de leitura

Se você estiver começando do zero, siga esta sequência:

1. [Categorias de Estoque](/cadastros/estoque/categorias)
2. [Produtos](/cadastros/estoque/produtos)
3. [Entrada de Material](/estoque/entrada-de-material)
4. [Notas Fiscais de Compra](/estoque/notas-fiscais-de-compra)
5. [Ordens de Separação](/estoque/ordens-de-separacao)

## Observações

- O fluxo de entrada de material tem dois caminhos e ambos convergem no mesmo objetivo: deixar saldo disponível para separação.
- A documentação foi estruturada para permitir leitura por etapa, sem uma página única longa.
- Os screenshots sugeridos ficam distribuídos por página, sempre com capturas limpas do demo.

## Resultado esperado

Ao seguir as páginas desta trilha, o operador consegue:

- criar os cadastros-base;
- registrar a entrada do material;
- separar itens para um técnico;
- confirmar a entrega final do material.
